# ConvertImages

A Python project for image conversion.

## Setup

1. Ensure Python is installed (version 3.7+ recommended).
2. The virtual environment is already configured in `.venv`.
3. To activate the virtual environment, run:
   - On Windows: `.venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the project: `python main.py`

## Usage

Modify `main.py` to add your image conversion logic.
